## 🚂 Getting Started 🚂

To get this project up and running locally on your pc, follow the steps:

### Pre requisites

- npm
  ```sh
   npm install npm@latest -g
  ```

### Installation

1. Clone the repo
  ```sh
   git clone https://github.com/fescherer/blog.git
  ```
2. Install NPM packages
  ```sh
   npm install
  ```
